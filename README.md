# Tainá Doces - Brigaderia Gourmet

Site institucional da Tainá Doces, especializada em brigadeiros artesanais gourmet.

## Tecnologias

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como executar o projeto

### Pré-requisitos

- Node.js instalado (recomendado usar [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Instalação

```sh
# Instalar as dependências
npm install
```

### Desenvolvimento

```sh
# Iniciar o servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:5173`

### Build

```sh
# Criar build de produção
npm run build

# Criar build de desenvolvimento
npm run build:dev
```

### Preview

```sh
# Visualizar o build de produção
npm run preview
```

### Linting

```sh
# Executar o linter
npm run lint
```

## Estrutura do Projeto

```
src/
├── assets/          # Imagens e recursos estáticos
├── components/      # Componentes React
│   └── ui/         # Componentes de UI (shadcn)
├── hooks/          # React Hooks customizados
├── lib/            # Utilitários
├── pages/          # Páginas da aplicação
└── main.tsx        # Ponto de entrada
```
